# DECISION-20260628-012: Approve ADR-019 Native vs. PWA-First Distribution

**Date:** 2026-06-28  
**Proposal:** Lead with a lightweight PWA/web landing quiz for discovery and first-time use; use the Flutter native app for engaged repeat users.  
**Risk Level:** High  
**Final Decision:** Approved

## Council Deliberation

## Rationale

The target learner faces install friction and storage constraints that a native-only strategy cannot fully address. A scoped PWA landing page lowers acquisition cost while preserving the existing Flutter app for engaged users.
